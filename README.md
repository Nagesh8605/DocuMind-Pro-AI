Ask anything from your documents using AI.

DocuMind Pro AI allows users to upload PDF, DOCX, or TXT files and ask questions directly from the document. The system retrieves relevant answers using semantic search powered by embeddings and vector databases.

## Problem Solved
Finding information in large documents is time-consuming. This project converts static documents into an interactive AI-powered knowledge assistant.

## Features
- Upload PDF, DOCX, TXT
- Ask questions from document
- AI-based semantic search
- Chat interface
- Copy answers
- Download answer as PDF
- Export full chat conversation
- Adjustable answer depth
